# Task: Create Property Details Page

A comprehensive page to view details of a specific real estate unit or plot.

**DESIGN SYSTEM (REQUIRED):**
Build using a premium Teal and Navy design system. Use color #00a99d for primary accents and #1a2b3c for dark headings. All cards should have a 32px border radius and be styled with a clean #fafbfc background. Typography should be Inter, with italicized subheadings. Use fully rounded pill-shaped buttons with bold uppercase text.

**Page Structure:**
1. Hero section with image gallery of the property
2. Main details card (Price, Size, Location, Status)
3. Features & Amenities icons grid
4. Installment plan calculator card
5. Contact agent / Book now form
6. Map view of the location
